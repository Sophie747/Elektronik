# Differenzialrechnung

| konstante Funktion | $f(x)=c$                    | $f'(x)=0$              |
| Potenzfunktion     | $f(x)=x^{n},n\in\mathbb{R}$ | $f'(x)=n\cdot x^{n-1}$ |
| Exponentialfunktion                    |                             |                        |
